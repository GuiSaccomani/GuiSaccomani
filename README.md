# 👨🏻‍💻 Guilherme Saccomani

**`Desenvolvedor FullStack`**

Me chamo Guilherme Saccomani, tenho 20 anos e sou apaixonado por tecnologia, desenvolvimento de software e criação de soluções digitais. Atualmente, curso **Engenharia de Software na FIAP**, onde venho aprofundando meus conhecimentos em desenvolvimento FullStack, arquitetura de sistemas, banco de dados e desenvolvimento de aplicações modernas.

Tenho experiência com desenvolvimento web utilizando tecnologias como React, Next.js, TypeScript, Node.js e Java, além de trabalhar com APIs, bancos de dados e interfaces responsivas. Também gosto de criar projetos criativos, sistemas personalizados e ideias inovadoras voltadas para startups e produtos digitais.

Sempre busco evoluir meus conhecimentos, aprender novas tecnologias e transformar ideias em projetos reais.
